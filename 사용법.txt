사용법

1. MySQL이나 MariaDB를 설치한다.

2. test라는 테이블을 만든 뒤 첨부되어있는 SQL파일을 test 데이터 베이스에 등록한다. 
(최초 설치시 계정을 등록하는데 계정을 등록할때 id : root, pass : 1234로 등록을 하거나 
본인이 하고싶은 것으로 설정후 서버의 ChatClientManager.java에서 아이디와 비밀번호를 수정하면된다.)

3. 서버 프로젝트를 먼저 실행한다. (IP 주소를 본인의 주소로 적어야함, PORT는 본인이 하고싶은 번호로한다 (8888추천))

4. 접속하고자 하는 클라이언트는 서버의 IP주소와 PORT 번호를 올바르게 적고 입장한다.

4. 혹 오류가 뜨거나 갑자기 멈추는 경우 서버와 클라이언트를 껏다 켜주기 바란다 (원인 불명 오류들임)
